# Context: Scope Creeper 🕹️

## Vision
Scope Creeper is designed for developers who want to escape the "boring software" trap. It uses agentic thinking to intentionally "disrupt" project linearity, pushing developers toward high-leverage, non-linear features while maintaining a diagnostic eye on implementation reality.

## Ecosystem Integration
- **Ditto (Ingest):** Feeds the engine with raw data and knowledge.
- **LootLens (Analyze):** Provides the metrics and tracking foundation.
- **Scope Creeper (Disrupt):** Mutates the plan into something memorable.

## Target Aesthetic
**"Tactical Retro Arcade"**
- **Reference Site:** [ranger-beers.com](https://ranger-beers.com)
- **Vibe:** High-contrast, grain, scanlines, digital tentacles, and "crazy prompt" memes.
- **Philosophy:** Developers love tactile, high-fidelity lo-fi design. It separates the "vibe coders" from the corporate templates.

## The Diagnostic Rubric
The AI evaluates:
1. **Boredom Threshold:** How much of this is just boilerplate?
2. **Expansion Gap:** What is the one architectural mutation that makes this project 10x more interesting?
3. **Timeline Risk:** Can this actually be built on local hardware/Apple Silicon?
