# Scope Creeper 🕹️💀

> **AI HALLUCINATES FLAWS. YOU HALLUCINATE FEATURES. LET'S FIND OUT WHO HAS THE BETTER DELUSION.**

Scope Creeper is a tactical diagnostic engine and agentic planning studio that gamifies the development process. It analyzes your repository, live site, or project pitch to calculate your **Delusion Meter**—the delta between your functional reality and your theoretical feature abyss.

## 📊 The Core Mechanics

### 1. The Delusion Meter
A scoring engine (0-100) that evaluates project viability:
- **0–30: The Code Corpse** (Sterile/Under-scoped)
- **31–70: The Sweet Spot** (Controlled Drift)
- **71–95: The Feature Abyss** (Aggressively Over-scoped)
- **96–100: Pure Delusion** (Timeline Extinction)

### 2. Reality vs. Illusion
- **Reality:** Scanned functional code, commits, and API routes.
- **Illusion:** Theoretical specs, wild prompts, and "creeper" mutations.

### 3. The Blueprint Studio
A three-column diagnostic interface to bridge the gap between your current codebase and your most insane feature delusions.

## 🚀 Quick Start (Local Dev)

```bash
# Clone the repo
git clone https://github.com/SipMyBeers/scopecreeper.git
cd scopecreeper

# Install dependencies
pnpm install

# Run the tactical dashboard
pnpm dev
```

## 🛠️ Tech Stack
- **Frontend:** Next.js (App Router), Tailwind CSS, Framer Motion
- **Monorepo:** pnpm workspaces
- **AI Engine:** Local-first (Ollama / LlamaIndex)
- **Aesthetics:** Tactical Retro Arcade (Onyx/Neon/Glitch)
